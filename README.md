# Docker Web App 🚀

## 📌 Description
This project shows how to run a simple Python Flask app using Docker.

## ⚙️ Tech Used
- Python
- Flask
- Docker

## 🐳 How to Run

Build:
docker build -t docker-web-app .

Run:
docker run -p 5000:5000 docker-web-app

## 🌐 Output
App runs on:
http://localhost:5000
